## Basic Usage

### Insert Images

和 Markdown 的图片插入方法一样，您可以使用以下语法在文章中插入图片：

```markdown
![Image alt](/path/to/image.jpg "Image title")
```
其中，图片的路径可以是 URL 或本地路径。如果是本地路径，图片应当放置在 `public/images` 目录下的合适位置，然后使用 `/images/` 开头的相对路径。

上述代码最终会被编译为：

```html
<img src="/path/to/image.jpg" alt="Image alt" title="Image title" />
```

下面是两个例子：

```markdown
![Image from Internet](https://picsum.photos/300/200?grayscale)
![Image from local](/images/posts/test/image-test/392-300x200-grayscale.jpg)
```
![Image from Internet](https://picsum.photos/300/200?grayscale)

![Image from local](/images/posts/test/image-test/392-300x200-grayscale.jpg)

### Image Caption

`Image title` 会作为图片的标题显示在图片下方。如果不设置（就像上面的两个例子），则会显示 `Image alt` 作为标题。
如果您不需要图片标题（例如一些装饰性图片），可以将 `Image title` 和 `Image alt` 两者都不设置。

下面是一个例子：

```markdown
![Image A](/images/posts/test/image-test/392-300x200-grayscale.jpg "Image B (rather than A)")
```

![Image A](/images/posts/test/image-test/392-300x200-grayscale.jpg "Image B (rather than A)")

## Advanced Usage

Markdown 或 MDX 的标准语法仅能支持基本的图片插入和标题显示。不过，我们可以在 MDX 中直接插入 JSX 代码，从而实现缩放、圆角、阴影等复杂的图片显示效果。

```jsx
// HTML img component
<img src={'/path/to/image.jpg'} alt={'Image example 1'} title={'Hello World'} width={200} height={300} />
// Next/Image component
<Image src={'/path/to/image.jpg'} alt={'Image example 2'} title={'Hello World'} width={200} height={300} />
```

上述组件也可以搭配其他 React/HTML 组件一起使用：

```jsx
<figure className={"max-w-2xl mx-auto"} >
  <img
    src={'/images/posts/test/image-test/104-600x400-grayscale.jpg'}
    alt={'Image example 1'}
    title={'Hello World'}
    className={"mx-auto rounded-3xl w-full h-auto blur-sm hover:blur-0"} />
  <figcaption className={"text-right text-sm text-destructive"} >Image example 1</figcaption>
</figure>
```
